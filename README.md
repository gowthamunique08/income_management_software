# 💰 Income Management Software

A desktop-based **Income Management Software** built using **Python** and **PyQt6** to help users efficiently track, manage, and analyze their income records. This application provides a user-friendly graphical interface for maintaining financial information in an organized way.

## 🚀 Features

- 📊 Add and manage income details
- ✏️ Edit and update income records
- 🗑️ Delete unnecessary records
- 🔍 Search and filter income entries
- 📈 Track income information efficiently
- 🖥️ User-friendly GUI with PyQt6
- 💾 Database integration for data storage
- ⚡ Fast and responsive desktop application

## 🛠️ Tech Stack

- **Programming Language:** Python  
- **GUI Framework:** PyQt6  
- **Database:** SQLite (if used)  
- **IDE:** VS Code  
- **Version Control:** Git & GitHub

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/gowthamunique08/income_management_software.git
```

### 2️⃣ Navigate to Project Directory

```bash
cd income_management_software
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Application

```bash
python main.py
```
